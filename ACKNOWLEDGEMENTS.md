# Credits & Acknowledgements

SoundShade is built using native Swift and C++ APIs and leverages the following open-source resources:

* **m1ddc**: A command-line tool to control monitor brightness and volume using DDC/CI on Apple Silicon Macs. 
  (Released under the MIT License)
* **ProxyAudioDevice**: A lightweight CoreAudio HAL audio plug-in template used to bridge and route volume attenuation signals on macOS.
